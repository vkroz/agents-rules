**agentpack** is a Python CLI tool that manages AI agent configurations across multiple tools. It acts as a compiler: write rules once in a vendor-neutral format, and agentpack compiles them into tool-specific output files (`CLAUDE.md`, `.cursor/rules/`, etc.).

Refer to the documentation for more details: 
- Development guide: @DEVELOP.md
- Functional specification: @docs/functional-spec.md
- User guide: @docs/user-guide.md
